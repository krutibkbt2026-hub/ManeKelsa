# ManeKelsa

ManeKelsa is a hyperlocal Android application developed using Kotlin and Firebase to connect local workers and households for daily work opportunities.

## Features
- Worker Registration
- Local Worker Listings
- Firebase Realtime Database
- Firebase Storage
- Direct Calling Feature
- Worker Availability Status
- Kannada Language Support
- User-Friendly Interface

## Technologies Used
- Kotlin
- Android Studio
- Firebase
- XML Layouts
- RecyclerView

## Project Objective
The project aims to digitally connect local workers such as cleaners, gardeners, and helpers with nearby households, improving accessibility and self-employment opportunities.
